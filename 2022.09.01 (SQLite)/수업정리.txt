INSERT INTO 테이블명(컬럼명) VALUES (데이터)

* 테이블명/컬럼명/명령어 전부 대소문자 구분안함

* 제약조건 (Constraint)
- Unique : 유일해야함. 단 Null은 허용 
- Not Null : Null이면 안돼
- Primary Key (PK, Pkey) : Unique + Not Null
- Foreign Key (FK, Fkey) : 타 테이블의 값을 참고
- Check : 값을 범위를 지정

* DML (Data Manipulation Language)
- INSERT INTO 테이블명(컬럼명) VALUES (값)
- DELETE FROM 테이블명 
- UPDATE 테이블명 SET 컬럼명=값 
- SELECT 컬럼명 FROM 테이블명
